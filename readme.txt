=== Borrufa Widget ===
Widget: És un giny per a col·locar en el Sidebar del WP.
Contributors: La borrufa
Language: Català
Requires at least: 5.4
Tested up to: 5.9.1
Stable tag: 1.0.4


== Descripció ==


El plugin Borrufa-Widget, permet afegir un giny en el sidebar del tema que tinguis instal·lat en el WordPress amb un enllaç per a poder llegir les notícies publicades en La borrufa en temps real i actualitzades automàticament.


== Política de servei extern ==

Aquest plugin es un complement basat en un tercer com a servei

Política: https://laborrufa.com/condicions/
Privacitat: https://laborrufa.com/politica-de-privacitat/


== Instal·lació ==

1. Descarregar.
2. Descomprimir.
3. Pujar a la carpeta plugins de WordPress (wp-content / plugins)
4. Activa a través del panell d'administració.
5. Configurar les opcions que siguin del teu interès.
